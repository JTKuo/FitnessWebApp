# Fitness Tracker & Progress WebApp

一個專為個人訓練量追蹤、PR 紀錄與體態演變設計的現代化健身管理 Web App。

## 🌟 亮點功能

- **訓練日誌 (Workouts)**: 支援動作範本、組數快速複製、休息計時器以及即時容量計算。支援動作拖拉排序。
- **成就系統 (Achievements)**: 自動追蹤各種動作的 1RM 與最重紀錄。提供 PR 卡片拖拉分類功能，視覺化管理個人巔峰。
- **數據分析 (Analytics)**: 透過圖表追蹤最近 90 天的體重趨勢、各部位訓練量佔比以及單一動作進步曲線。
- **體態紀錄 (Body Stats)**: 整合 TDEE/BMR 計算機，支援三向（正面、側面、背面）歷史照片比對功能。
- **管理者模式 (Admin Mode)**: 支援管理者帳號切換視角，方便進行跨使用者數據檢視與維護。

## 🚀 技術架構

- **前端 (Frontend)**: 基於 Vanilla JS, CSS3, 與 HTML5 建立，保證極高的反應速度。
- **後端 (Backend)**: 使用 Google Apps Script (GAS) 作為 Serverless API，整合 Google Sheets 進行持久化存儲。
- **部署 (Deployment)**: Frontend 部署於 GitHub Pages，Backend 於 Google App Engine (GAS) 運行。

## 🛠️ 開發與設定

### 環境需求
- Python 3.9+ (用於維運腳本)
- PowerShell (Windows)

### 安裝步驟
1. 執行初始化腳本：
   ```powershell
   ./scripts/setup.ps1
   ```

2. 啟動開發環境：
   本專案前端為靜態網頁，可直接開啟 `frontend/index.html` 或使用 VS Code 的 Live Server。
